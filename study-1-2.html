바닐라 JS로 크롬 앱 만들기 (2) - Document Object And Event

목차
  • Document Object
  • Event
  • 필기

Document Object

Javascript를 사용하면서 우리는 동적인 페이지를 구성할 수 있다. 그러기 위해서 페이지에서 각 부분이 어떻게 컴퓨터가 인식하고 있는지 알고 있어야한다. 이 과정에서 먼저 HTML에 대한 이해도가 기반이 되어야 한다. Class와 Id 정도의 개념을 이해하고 있다면 괜찮다.

우리가 Document에 접근하기 위해서는 크게 2가지 방법을 사용한다.

  1. getElementById("타겟 Id")
    이 친구의 경우, 우리는 타겟 Id에 해당하는 모든 HTML element를 가져온다.
  2. queryselector("CSS 형식")
    이 친구의 경우, 위와 달리 class나 HTML 요소를 접근 할 수 있다. 다만 먼저 읽힌 1개만 가져오기 때문에 전부 가져오고 싶은 경우 queryselectorAll을 사용해야한다.

Event

웹 페이지에서 일어나는 상황을 Event라고 표현한다. 배운 내용에 한해서는 click, submit, change가 있다. 이 과정에서 우리는 addEventListener()이라는 함수를 사용한다. 앞서 가져온 Document Object에 변화가 일어나는지를 관찰하는 옵저버 같은 거라고 생각하면 편하다.

이 때, addEventListener() 함수의 경우 2개의 변수를 가진다.

  1. event
    아까 언급한 click, submit, change 같은 웹 페이지에서 일어나는 변화를 말한다.
  2. function
    해당 이벤트가 발생했을 때, 실행할 함수를 이야기한다.
